# About HotelByte Blog

This blog shares our technical journey building [HotelByte](https://hotelbyte.com), a next-generation hotel distribution platform.

## What We Write About

### Technical Deep Dives
- Go microservices architecture
- Hotel API aggregation patterns
- High-performance caching strategies
- Concurrency and parallel processing

### Industry Insights
- Hotel distribution business models
- BYOL (Bring Your Own License) vs traditional platforms
- Travel tech trends and predictions

### Developer Experience
- Supplier integration best practices
- API design principles
- Debugging and troubleshooting
- SDK development

## Our Stack

- **Backend**: Go 1.25+, go-zero
- **Frontend**: Vue 3, React 18
- **Database**: MySQL 8, Redis 6
- **Infrastructure**: Docker, Kubernetes
- **Monitoring**: Prometheus, Sentry, OpenTelemetry
