# HotelByte Official Blog

Technical articles about hotel API distribution, Go microservices, and building scalable travel tech platforms.

## Quick Start

This blog is built with Jekyll and hosted on GitHub Pages.

### Local Development

```bash
# Install dependencies
bundle install

# Start local server
bundle exec jekyll serve

# Open http://localhost:4000
```

### Adding a New Post

1. Create a new file in `_posts/` with the format `YYYY-MM-DD-title.md`
2. Add front matter at the top:
```yaml
---
layout: post
title: "Your Post Title"
date: 2026-02-04 18:30:00 +0000
categories: [category1, category2]
tags: [tag1, tag2]
author: "Author Name"
---
```
3. Commit and push
4. GitHub Actions will automatically deploy
